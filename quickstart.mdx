---

OWASP Nest is available at [nest.owasp.org](https://nest.owasp.org) with no installation required. This guide walks you through the core steps to get started as a community member.

<Steps>
  <Step title="Visit nest.owasp.org">
    Open [nest.owasp.org](https://nest.owasp.org) in your browser. The home page gives you immediate access to project search, chapters, contribution opportunities, and mentorship programs — no account required for browsing.
  </Step>

  <Step title="Sign in with GitHub">
    Click **Sign in with GitHub** in the top navigation bar. Nest uses GitHub OAuth for authentication, so no separate account creation is needed. Signing in enables personalized features such as saving interests and accessing mentorship programs.

    <Info>
      Nest requests only the minimum GitHub permissions required for authentication. Your repository and organization data are not accessed through your personal token.
    </Info>
  </Step>

  <Step title="Browse projects using the search interface">
    Navigate to **Projects** to explore the full catalogue of OWASP projects. Use the search bar to filter by keywords such as a technology, topic, or project name. You can sort results by activity, stars, forks, or contributor count, and filter by project level (Flagship, Production, Lab, Incubator).

    Each project card shows a summary, activity indicators, and top contributors. Click **View Details** to open the full project page with GitHub statistics, open issues, and AI-generated insights.
  </Step>

  <Step title="Find contribution opportunities">
    Navigate to **Contribute** to browse open GitHub issues across all active OWASP projects. Each issue card displays the project name, issue labels, creation date, and comment count.

    Click **Read More** on any issue card to open an AI-generated summary of the issue along with recommended steps to address it. Click **View Issue** to go directly to the issue on GitHub.
  </Step>

  <Step title="Set up NestBot on Slack">
    Join the [OWASP Slack workspace](https://owasp.org/slack/invite) and navigate to the [#project-nest](https://owasp.slack.com/archives/project-nest) channel. NestBot is already active in the workspace and responds to slash commands in any channel.

    Try a few commands to get started:

    ```
    /nest project zap
    /nest chapter london
    /nest contribute python
    ```

    <Tip>
      You can invite NestBot to any OWASP Slack channel by typing `/invite @NestBot`. Once added, team members can query projects and issues without leaving Slack.
    </Tip>
  </Step>
</Steps>
